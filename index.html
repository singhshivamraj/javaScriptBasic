JavaScript performance

This task demonstrates JavaScript event handling through three small interactive features:

Live Input Display: When the user types something into the input field, the same text is instantly displayed on the screen.
Addition of Two Numbers: The user enters two numbers. On clicking a button, their sum is calculated and displayed on the screen.
Background Color Change: When a button is clicked, the background color of the webpage changes.
